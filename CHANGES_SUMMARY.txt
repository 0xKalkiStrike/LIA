                            SUPPORT & DEBUGGING

COMMON ISSUES & SOLUTIONS:

Issue: Avatar not loading
→ Check LIA.vrm exists, or upload custom VRM in onboarding

Issue: No voice output
→ Enable browser permissions, check volume, try different voice

→ Verify audio is playing, check browser console (F12), clear cache

→ Start local Ollama (ollama serve), use smaller model

Issue: Gray/desaturated character
→ Refresh page, update graphics drivers, try different browser
