## 🔧 Troubleshooting

### "Character still looks grayscale"
→ Hard refresh: **Ctrl+Shift+R** (clear cache)

### "Voice still sounds robotic"
→ Install Piper TTS (see above for natural voice)

### "Animations not smooth"
→ Close other browser tabs, disable extensions

### "App won't start"
→ Check if port 8000 is available

### "Out of memory"
→ Reduce animation complexity in settings
